A glance at my CV begs the questions "Why the career change?", and "Why software development?". My career so far has been a rewarding and rich experience. However, I realised that what I was craving the opportunity to sharpen my focus and build depth in a technical skill. Simultaneously, I also understood that I wanted to do something creative.

Software development had never crossed my mind as a possible option. I had assumed, like many, that it was only for CS graduates and maths whizzes. As I was questioning my career choices I was encouraged by several friends and family members who work in the tech industry to scrutinise my assumptions. As I explored, I found that software development could satisfy my technical and creative appetite and, it was not only something that I can do, I also really enjoy it!

// PROJECTS

JavaScript Maze Game

Oh no! Tony the Hawk has broken his board. Help him get back to his nest so he can repair it. Watch out for predators along the way! This was my first collaborative project that utilised DOM manipulation, event listeners, callback handlers and modals built with Vanilla Javascript. A group project at iO managed with Git and GitHub.


React Invoicing App

Built a mobile-first front end in component-based React.js for a furniture store application that consumed an existing API. The application implemented features such as pagination, search and filters and responsiveness. FE wireframe was provided by 'the client'. Utilised Bootstrap 5.2.2 as a CSS framework. A group project at iO managed with Git and GitHub.


API Build for Furniture Store

Built an OOP based API for a pre-provided FE furniture store application. Wrote an SQL script to construct a database from an existing JSON file. The API was built from a pre-defined API documentation. OOP code was written in accordance with SOLID principles and utilised design patterns such as singleton, data access object and PHP entity hydration. Designed and ran PHP unit testing on the application. A group project at iO managed with Git and GitHub.


iO Academy Portal

A group project at iO Academy where our cohort were given a prioritised backlog of stories and tasks that included new programmes and bug fixes. The Academy Portal is built with an MVC framework. Throughout this project I primarily worked in JavaScript fixing error message consistency and also had to re-code all legacy validations on form inputs in addition to creating new ones.


OOP Blackjack

Liz Truss and Rishi Sunak battle it out in this OOP version of blackjack written in PHP. This programme was re-compiled into an OOP programme from a functional version that I had previously created. This OOP programme makes use of classes, interfaces and implementation. The game accounts for draw, lose-lose, and win-lose scenarios. I plan to continue this project by creating a more engaging user interface in the front end.
